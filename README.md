# Instagram Clone 

This repository holds the code for an Instagram clone. It was built using:
<li> REACT.js </li>
<li> Next.js </li>
<li> Tailwind CSS </li>
<li> Firebase(V9) </li>
<li> NextAuth </li>
<li> Recoil </li>
<li> TypeScript </li> 
<li> Faker.js </li>
